# DL4Bio — Assignment Template
Este repositorio será utilizado para la resolución de los Trabajos Prácticos (TPs).

## Estructura
- `notebooks/` → notebook principal del TP.
- `results/` → outputs generados (figuras, tablas, métricas).
- `environment.yml` → entorno reproducible del curso.

## Uso del entorno (recomendado conda / miniconda) y el archivo environment.yml
Una vez activado el entorno, podés trabajar con el editor o entorno que prefieras:
- Jupyter Notebook / Jupyter Lab,  VSCode (otro IDE), Colab etc.

# Requisitos del TP
El notebook debe poder ejecutarse de principio a fin sin errores utilizando el entorno definido en `environment.yml`.
No debe haber celdas ejecutadas fuera de orden.
Todos los outputs deben guardarse explícitamente en results/.

## Evaluación
Se evaluará:
Reproducibilidad completa.
Corrección técnica del análisis.
Claridad en la explicación.
Organización del repositorio.
Uso progresivo y coherente de commits (no commit único final).

El repositorio debe permitir que cualquier evaluador lo clone y reproduzca los resultados sin modificaciones.
